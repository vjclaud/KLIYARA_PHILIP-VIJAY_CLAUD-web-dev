Discover Movies

What does this web application do?
It helps you discover new movies

How is this any different from finding popular movies on sites like imdb.com or even a simple Google search?
While a Google search can easily find the best comedies or a list of critically acclaimed movies, the results will almost always include movies that you have already watched. This web application remembers every movie that you like, dislike or add to a watchlist and those movies will never be suggested again

List of cool features
  • Filter suggestions by genre, release date, release period, keywords etc.
  • Maintain lists of movies that you like, dislike or the ones that you would like to watch later
  • Find and Follow people with similar tastes for easy access to their movie lists
  • Write reviews and see what other people have to say in the 'reviews' section of each movie
  • Search for movies by title and add them to your list
